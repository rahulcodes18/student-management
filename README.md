# Student Management System

## Tech Stack
- Spring Boot
- MySQL
- REST API
- Maven

## Features
- Add Student (POST)
- Get All Students (GET)
- Update Student (PUT)
- Delete Student (DELETE)

## How to Run
1. Clone repository
2. Import in VS Code / IntelliJ
3. Configure MySQL database
4. Run Spring Boot application
5. Test APIs using Postman
